# WiseClick Finance

> 12 free online financial calculators. No sign-up required.

**Live site:** [calc.wiseclick.site](https://calc.wiseclick.site)

## Calculators

- Mortgage Calculator
- Loan Calculator
- Compound Interest Calculator
- Credit Card Payoff Calculator
- Auto Loan Calculator
- Retirement Calculator
- 401(k) Calculator
- Tax Calculator
- Annuity Calculator
- Savings Calculator
- Paycheck Calculator (all 50 US states)

## Features

- All calculations run in your browser — your financial data never leaves your device
- Amortization schedules for mortgage and loan calculators
- State-specific tax calculations for paycheck calculator
- JSON-LD structured data (WebApplication + FAQPage schemas)
- Mobile responsive

## Tech

- Pure HTML/CSS/JavaScript
- Hosted on GitHub Pages
- Zero dependencies
